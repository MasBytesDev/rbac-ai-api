### RBAC API

A modular Role-Based Access Control (RBAC) API built with Java 21 and Spring Boot.
Provides complete CRUD operations for roles, permissions, and users, with domain-driven exceptions, DTOs, and centralized error handling.
This version establishes the foundation for secure authentication and authorization, evolving towards AI-powered features in future releases.
